# Privacy Policy

Last Updated: February 8, 2026

## Introduction

WiFi Alert (the "App") is an application designed to detect Wi-Fi disconnections and send notifications to prevent unintended mobile data usage. This Privacy Policy explains what information the App collects and how it is used.

## Information We Collect

### Location Information
The App uses location information to enable Wi-Fi monitoring at registered locations:

- **GPS Location**: Used when registering locations and detecting entry/exit from registered areas
- **Purpose**: Geofencing technology monitors Wi-Fi only when you are within a specified location range
- **Storage**: All location data is stored only on your device and is never transmitted to external servers

### Network Connection Information
- **Wi-Fi Connection Status**: Monitors whether Wi-Fi is connected or disconnected
- **Wi-Fi Name (SSID)**: Retrieves the name of the connected Wi-Fi network (for notification display)
- **Purpose**: To detect Wi-Fi disconnections and send notifications
- **Storage**: Device only

### Local Storage
- **Registered Location Data**: Location name, latitude/longitude, monitoring radius, etc.
- **Storage**: Saved only in your device's local storage (AsyncStorage)
- **External Transmission**: None

## How We Use Information

Collected information is used only for the following purposes:

1. **Wi-Fi Disconnection Detection and Notification**: Send notifications when Wi-Fi is disconnected at registered locations
2. **Location-Based Monitoring**: Enable Wi-Fi monitoring only within registered location ranges
3. **App Functionality**: Display connection status, manage monitoring, etc.

## Information Sharing

**The App does NOT share any collected information with third parties.**

- Advertising networks: None
- Analytics services: None
- External server transmission: None

All data is stored only on your device and is never transmitted externally.

## Data Retention Period

- **Location Information**: Stored on device until you delete the location
- **Settings**: Stored until you uninstall the app
- **Temporary Network Information**: Not stored after notification is sent

## Your Rights

### Data Deletion
- You can delete registered locations individually from the "Locations List" screen in the app
- Uninstalling the app will completely delete all data

### Permission Management
- You can change location and notification permissions anytime from iOS Settings > WiFi Alert
- Disabling permissions will disable corresponding features

## Required Permissions

The App uses the following permissions:

### Location Permission (Required)
- **Purpose**: To determine if you are within a registered location range
- **Types**: Foreground location, Background location

### Notification Permission (Required)
- **Purpose**: To send notifications when Wi-Fi is disconnected

### Network State Access (Automatic)
- **Purpose**: To monitor Wi-Fi connection status

## Security

The App implements the following measures to protect your data:

- All data stored in device local storage (no external transmission)
- Uses iOS standard data protection features
- Does not request unnecessary permissions

## Children's Privacy

The App is not directed to children under 13 years of age. We do not knowingly collect personal information from children under 13.

## Changes to Privacy Policy

This Privacy Policy may be updated as needed. If there are significant changes, we will notify you through the app or App Store description.
